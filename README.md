# HOA Drive-By

SwiftUI iPhone MVP for HOA/property violation inspections.

## Features
- Community and property management
- New violation capture
- Camera or photo library intake
- GPS stamp support
- Violation status tracking
- PDF report export
- Local JSON persistence
- CSV property import

## Open in Xcode
1. Unzip the archive.
2. Open `HOADriveBy.xcodeproj`.
3. Set your Apple Team in Signing & Capabilities.
4. Build and run.

## Permissions included
- Camera
- Photo Library
- Location When In Use

## Notes
- Camera works best on a physical iPhone.
- Data is local-only for this MVP.
